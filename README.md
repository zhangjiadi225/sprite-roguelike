# 精灵肉鸽 (Sprite Roguelike)

一个像素风格的肉鸽类宝可梦游戏。

🎮 **在线体验：** [即将部署到 Vercel]

## 📖 项目介绍

这是一个结合了 Roguelike 元素和宝可梦玩法的像素风格游戏。玩家将在随机生成的地图中探险，捕捉和培养各种精灵，体验每次都不同的冒险旅程。

### ✨ 核心特色

- 🧬 **精灵融合系统** - 独特的肢体继承机制，融合出独一无二的精灵
- ⚔️ **回合制战斗** - 经典的回合制战斗，属性相克策略
- 🎲 **随机地图** - 每次游戏都有不同的地图布局和房间
- 🎯 **捕捉系统** - 捕捉野生精灵，组建强大队伍
- 🛒 **商店系统** - 购买道具，提升战斗能力
- 💾 **局外系统** - 死亡后可带出精灵，持续成长

## 🎮 游戏玩法

1. **探索地图** - 在随机生成的地图中探险
2. **战斗** - 与野生精灵战斗，获得经验和金币
3. **捕捉** - 使用精灵球捕捉喜欢的精灵
4. **融合** - 融合两只精灵，创造更强大的新精灵
5. **购物** - 在商店购买药水、精灵球等道具
6. **局外** - 死亡后带出精灵，下次游戏继续使用

## 🛠️ 技术栈

- **框架**: Next.js 15
- **游戏引擎**: Phaser 3
- **语言**: TypeScript
- **构建工具**: Vite
- **部署**: Vercel

## 🚀 本地运行

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```
